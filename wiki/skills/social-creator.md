# Social Creator - 社交媒体爆款内容创作工具

![Status](https://img.shields.io/badge/status-active-success)
![ClawHub](https://img.shields.io/badge/Platform-ClawHub-orange)

> AI 行业洞察 + 爆款文案创作 + 视觉设计，一键生成小红书/抖音/视频号高互动内容

---

## 技能定位

**核心价值**：复合型自媒体内容专家，具备三重核心能力：
- 🧠 AI 行业洞察：联网搜索实时掌握最新产品动态
- 📱 爆款文案创作：创作高互动率的一句话式文案
- 🎨 视觉设计能力：生成小红书友好的 3:4 视觉页面

**适用场景**：
- 小红书/抖音/视频号内容创作
- AI 科技主题文案生成
- 产品对比评测内容
- 工具推荐类笔记

## 核心能力

### 实时资讯获取（前置步骤）

创作文案前**必须**联网搜索最新信息：

**搜索范围**：
- 🔥 热门模型：GPT5、Claude3.7、Kimi2.5、MiniMax2.7、Qwen3.5-plus 等
- 🆕 新发布产品：近 10 天内发布的新模型/新工具
- 💻 AI Coding 工具：ClaudeCode、Cursor、Windsurf、OpenClaw 等
- 🤖 Agent 产品：OpenClaw、WorkBuddy、Qclaw、CoPaw、ArkClaw 等
- 📊 行业热点：当前 AI 圈热议话题

**事实准确性要求**：
- ✅ 使用真实存在的产品/模型名称
- ✅ 版本信息需准确（通过搜索确认）
- ✅ 避免编造不存在的功能对比
- ⚠️ 无法确认时标注"请核实后使用"

## 文案创作核心原则

### 文案策略类型

| 类型 | 策略 | 示例 |
|------|------|------|
| 对比争议型 | 两款热门产品对比 | "A 还是 B？哪个更好用？" |
| 求教互动型 | 以请教姿态引发分享欲 | "求用过的大佬说说！" |
| 体验分享型 | 强调"真实体验" | "真实体验下来..." |
| 时效热点型 | 绑定最新发布 | "新出的 XX 真有那么神？" |

### 输出要求

- **数量**：1-5 条（默认 3 条）
- **长度**：每条 15-50 字
- **格式**：纯文本 + 5 个#话题标签
- **风格**：口语化、带情绪、有悬念

### 话题标签规则

每条文案末尾追加 **5 个热门#话题**：
- 2 个泛流量话题（如 #AI #科技）
- 2 个垂直领域话题（如 #AI 工具 #效率神器）
- 1 个精准长尾话题（如 #OpenClaw #AI 编程）

### 互动引导技巧

互动引导语必须**单独一行**：
- ❓ 疑问结尾："哪个更好用？"
- ⏰ 时效压迫："在线等！"
- 🙏 求教姿态："求大佬指点"
- 💬 邀请讨论："大家怎么看？"

## 视觉页面生成

### 核心设计理念

> **少即是多**：元素极简，突出文案内容本身，让文字成为视觉焦点

### 页面规格

| 属性 | 要求 |
|------|------|
| 比例 | **3:4**（viewport 单位 + aspect-ratio） |
| 格式 | 单文件 HTML（内联 CSS） |
| 数量 | 每条文案生成一个独立页面 |
| 适配 | ✅ 移动端完美适配，F12 手机模式无背景缝隙 |

### 设计规范

| 元素 | 要求 |
|------|------|
| 背景 | 纯色或简单渐变，无图案无纹理 |
| 主色调 | 科技蓝紫/黑白极简/小红书红 |
| 字体 | 系统默认无衬线字体，文案加粗 |
| 文案字号 | 36-48px（占据视觉中心） |
| 关键字 | 对比色/背景色块/下划线高亮 |
| 留白 | 四周留白≥15% |

### 禁止元素

- ❌ "干货分享""学习笔记"等标签
- ❌ "点赞""收藏""关注"等引导
- ❌ 多条文案堆砌
- ❌ 复杂边框/花纹/图案
- ❌ 底部互动引导区

## 输出格式

### 完整输出结构

```markdown
## 🔍 实时资讯摘要
（搜索到的最新热点，作为文案创作依据）

## 📝 爆款文案列表（3-5 条）

### 文案 1
[正文案内容]

[互动引导语]

#话题 1 #话题 2 #话题 3 #话题 4 #话题 5

## 🎨 视觉页面文件
输出目录：`output/{任务名称}/`
- cover-1.html
- cover-2.html
- cover-3.html
```

## 使用示例

### 文案示例

**示例 1**：
```
正文：Claude 3.5 和 GPT-4o 写代码，真实体验差距有多大
引导：求用过的大佬说说！
话题：#AI #大模型 #Claude #GPT4o #AI 编程
```

**示例 2**：
```
正文：Cursor 还是 Windsurf？AI 编程工具选哪个不踩坑
引导：在线等！
话题：#AI 工具 #编程神器 #Cursor #Windsurf #效率提升
```

### 视觉风格参考

- 背景：线性渐变 + 圆角卡片容器（40px border-radius）
- 文案：42px 加粗，居中，行间距 1.8
- 关键字：块级高亮（inline-block），圆角 12px
- 互动引导：独立区块，与正文分离，圆角 20px 背景框
- 装饰：文案前加 1 个相关 emoji

## 截图方式

**方式 1：电脑浏览器 + F12 手机模式**（推荐）
1. 浏览器打开 HTML 文件
2. 按 F12 打开开发者工具
3. 点击"切换设备工具栏"（Ctrl+Shift+M）
4. 选择任意手机型号
5. 直接截图

**方式 2：电脑浏览器 + 手动调整窗口**
- 调整浏览器窗口为 3:4 比例
- 直接截图

**方式 3：手机浏览器**
- 将 HTML 发送到手机
- 手机浏览器打开并截图
