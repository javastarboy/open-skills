# 技能使用指南

> Open Skills 各技能的详细使用方法、参数配置、输出示例

## Blog Rewriter 使用方法

### 基本用法

**方式 1：链接输入**
```
请帮我改写这篇文章：https://example.com/article
```

**方式 2：文本输入**
```
请帮我改写这段内容：
[粘贴原文文本]
```

---

### 参数配置

#### 自定义引流话术

提供你的固定引流话术：
```
请帮我改写这篇文章：[链接]

我的引流话术：
📌 关注【我的公众号】，对话框点击"AI 交流"
🔗 关注【我的公众号】，对话框点击"学习专栏"免费领取知识库
👉 添加微信：LHYYH0001
```

#### 文章类型指定

指定文章类型影响改写风格：
```
文章类型：技术干货
```

可选类型：
- 技术（专业、严谨）
- 情感（温暖、共鸣）
- 资讯（客观、清晰）
- 干货（实用、结构化）
- 故事（生动、叙事）

---

### 输出结果

**改写文案**：完整改写后的文章，包含：
- 吸引人的开头
- 清晰的小标题
- 图片占位标记
- 你的引流话术
- 互动引导语

**HTML 封面**：1200x900 px（4:3 比例）

**5 个备选标题**：
```
1. 【悬念型】...
2. 【数据型】...
3. 【反问型】...
4. 【利益型】...
5. 【故事型】...
```

**改写报告**：
```
- 原文字数：XXX
- 改写后字数：XXX
- 相似度估计：<30%
- AI 化处理：已执行
- 事实核实：X 处已联网核实
```

## Social Creator 使用方法

### 基本用法

**方式 1：指定主题**
```
请生成 3 条 AI 编程工具对比的爆款文案
```

**方式 2：自动主题**
```
请生成 5 条爆款文案
```

（自动联网搜索 AI 圈最新热点）

---

### 参数配置

#### 文案数量
```
文案数量：3 条
```

#### 主题方向
```
主题方向：AI 编程工具 / 大模型对比 / Agent 应用
```

#### 视觉风格
```
视觉风格：科技感 / 简约风 / 深色模式
```

---

### 输出结果

**3-5 条爆款文案**：
```markdown
### 文案 1
正文：Claude 3.5 和 GPT-4o 写代码，真实体验差距有多大
引导：求用过的大佬说说！
话题：#AI #大模型 #Claude #GPT4o #AI 编程
```

**HTML 封面文件**：
- cover-1.html
- cover-2.html
- cover-3.html

每个文案一个独立 3:4 页面。

## AI News Xiaohongshu 使用方法

### 方式 1：OpenClaw 主流程调用（推荐）

由 OpenClaw 主流程处理：
```
请生成今日 AI 资讯小红书内容
```

OpenClaw 会自动：
1. 使用 web_search 搜索最新资讯
2. 调用脚本生成文案和 HTML

---

### 方式 2：脚本直接运行（演示模式）

```bash
cd ~/.openclaw/workspace/skills/ai-news-xiaohongshu
node scripts/create-xiaohongshu-content.js
```

使用内置演示数据生成示例内容。

---

### 参数配置

编辑 `references/user-config.md`：

```markdown
# 引流话术
📌 关注我，每日更新 AI 前沿资讯
👉 评论区留言"资料"获取 AI 工具包

# 核心资讯数量
5

# HTML 屏数
2
```

---

### 输出结果

输出目录：`output/日期-序号/`

**xiaohongshu-copy.md** - 小红书文案：
```markdown
🔥 AI 圈炸了！OpenAI 连夜发布新模型，性能提升 300%！

家人们谁懂啊！今天 AI 圈又是大动作不断😱...

📌 今日核心资讯：
• OpenAI 发布 GPT-4.5，推理速度提升 3 倍🚀
• 阿里千问开源新模型...

👇 想要 AI 工具包的宝子评论区扣"666"
```

**cover.html** - 3:4 比例 HTML 封面（1-3 屏可滑动）

**news-summary.md** - 资讯汇总表格

**sources.md** - 原始来源链接

## 通用使用技巧

### 截图方式

**电脑浏览器 + F12 手机模式**（推荐）：
1. 浏览器打开 HTML 文件
2. F12 → 切换设备工具栏
3. 选择手机型号
4. 截图

### 文件管理

输出文件自动保存到：
```
~/.openclaw/workspace/skills/skill-name/output/YYYY-MM-DD/
```

脚本执行完成后自动打开访达窗口。

### 配置修改

编辑技能目录的配置文件：
```
skill-name/references/user-config.md
```

## 常见问题

### Q: Blog Rewriter 改写后相似度如何？

**A**: 目标相似度<30%，通过句式重组、词汇替换、结构调整实现。

### Q: Social Creator 如何保证文案时效性？

**A**: 创作前必须联网搜索最新产品信息，确保事实准确。

### Q: AI News Xiaohongshu 如何获取真实数据？

**A**: 需要由 OpenClaw 主流程使用 web_search 搜索，然后传入脚本生成内容。
