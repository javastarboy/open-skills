## 📖 项目简介

`open-skills` 是一个基于 ClawHub 生态构建的开源技能仓库，开发并提供实用、高效的 AI 技能，覆盖多种应用场景。

所有技能完全开源免费，持续迭代更新，欢迎社区贡献与交流！

---

## ✨ 项目特点

| 特点 | 说明 |
|------|------|
| 🆓 **开源免费** | 所有技能代码公开，自由使用 |

| 🤝 **欢迎贡献** | 支持社区提交技能或改进建议 |

## 🚀 快速开始

### 前置条件

- 已注册并登录 [ClawHub](https://clawhub.ai)
- 了解 ClawHub 技能基本使用方式

### 使用步骤

1. **浏览技能** - 在本仓库查看感兴趣的技能
2. **下载/导入** - 通过 ClawHub 平台导入对应技能
3. **配置使用** - 按技能说明进行参数配置
4. **开始创作** - 享受 AI 带来的效率提升

### 获取方式

```
✅ 所有技能已上架 ClawHub 平台
✅ 直接在 ClawHub 搜索技能名称即可使用
✅ 源码可在本仓库查看学习
```

## 🤝 如何贡献

欢迎以以下方式参与贡献：

- 🐛 **提交 Issue** - 反馈问题或提出新技能建议
- 🔧 **提交 PR** - 修复问题或新增技能
- 📢 **分享传播** - 推荐给更多需要的朋友
- ⭐ **点亮 Star** - 支持项目持续更新

### 贡献流程

```
1. Fork 本仓库
2. 创建新分支 (feature/your-skill-name)
3. 提交更改
4. 发起 Pull Request
```
